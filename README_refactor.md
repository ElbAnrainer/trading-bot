# Trading Bot – Refactor Stand

## Überblick

Dieses Projekt ist ein reines **Simulations- und Analyse-System** für Aktien.

Es gibt:

- **keine Broker-Anbindung**
- **keine echten Orders**
- **keine Anlageberatung**

Das System kombiniert aktuell:

- klassische Analyse und Backtesting
- selbstlernenden Score
- Portfolio-Gewichtung
- Risk-Management
- Walk-Forward-Analyse
- Mini-Trading-System mit Portfolio-Persistenz
- Live-Dashboard im Terminal
- realistische 10.000-€-Backtest-Schätzung
